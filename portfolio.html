GeminEye – AI Navigation for the Blind

📖 Overview:
GeminEye is an AI-powered assistant that helps blind users navigate safely by detecting obstacles and signs in real time. It provides spoken feedback and responds to voice commands, making it highly accessible and intuitive to use.

📊 Functional Flowchart:

Start
Camera Feed Activated
Real-Time Object Detection
Obstacle or Sign Identified?
No
Continue Scanning
Yes
Use Gemini API + OCR
Convert to Audio Feedback
Deliver Spoken Guidance to User
Voice Command Detected?
Yes
Process Command + Respond
No
Loop Back to Camera Feed
End (Manual Stop)

🚀 Future Enhancements:

  • GPS navigation for outdoor use
  • Wearable edge-device deployment
  • Multilingual voice feedback

  • Emergency SOS activation

AI Slide-to-Quiz Generator

📖 Overview:
This project converts lecture slides (PDF or PPTX) into interactive quizzes using AI. It extracts key concepts from each slide and formulates multiple-choice questions automatically, turning static material into an engaging, testable format for students and educators.

📊 Functional Flowchart:

User Uploads Slide (PDF/PPTX)
Extract Slide Text
Parse Concepts from Each Slide
Generate Questions using Gemini API
Format into MCQs
Display Quiz Interface
User Takes the Quiz
Show Score & Feedback

🚀 Future Enhancements:

MyFridge – Smart Meal Generator

📖 Overview:
MyFridge uses AI to turn fridge photos into recipe suggestions. It scans images to detect ingredients, identifies what's available, and generates personalized meals – helping reduce food waste and simplify meal planning.

📊 Functional Flowchart:

User Uploads Fridge Image
Image Processed via Gemini Vision API
Ingredients Extracted from Image
Check Ingredient Validity
Pass Items to Meal Generator
AI Suggests Recipe Ideas
User Sees Meal List & Instructions

🚀 Future Enhancements:

  • 1. Barcode + expiration detection via camera
  • 2. Nutritional analysis and calorie breakdown
  • 3. Smart pantry tracking (learn what you usually stock)
  • 4. Recipe rating and user feedback system
  • 5. Voice assistant integration
